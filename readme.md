# ASUS FX73VD BIOS: FX753VD 
1. hackintosh opencore EFI
2. support Mac big sur 11, Tahoe 26

driver info：
1. iGPU：support
2. network：support
3. audio: support
4. sleep: support
5. touchpad：
    1）loop support
    2）GPIO not support
6. wlan：not support
7. bt：not support
8. FN keys：support，exclude F2 / F8
9. keyboard backlight：support


# install 
1. install OC/installFnKey.pkg to make fn work
2. install OC/installWakeSync.pkg to make wakeup active sound and time sync (Optional)
